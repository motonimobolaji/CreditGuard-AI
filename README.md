# CreditGuard AI

CreditGuard AI is an end-to-end credit card fraud detection and risk analytics platform designed to identify suspicious transactions, generate fraud risk scores, and support fraud investigation workflows.

## Project Overview

This project uses credit card transaction data to detect fraudulent activity using machine learning and explainable AI. The platform will include data cleaning, exploratory data analysis, feature engineering, model training, SQL storage, Power BI dashboards, and a Streamlit web application.

## Business Problem

Credit card fraud creates financial losses for banks, merchants, and customers. Fraud teams need tools that can identify high-risk transactions quickly while reducing unnecessary manual reviews. This project simulates how a financial institution could use machine learning and analytics to prioritize suspicious transactions.

## Dataset Features

The dataset includes transaction-level features such as:

- Transaction amount
- Transaction hour
- Merchant category
- Foreign transaction indicator
- Location mismatch indicator
- Device trust score
- Transaction velocity in the last 24 hours
- Cardholder age
